# 社交型LLM聊天机器人中的信任：作为情境化用户状态的动态演化

- **来源**: arXiv 预印本
- **日期**: 2026年
- **链接**: https://arxiv.org/pdf/2604.22417
- **类目**: 社会与人际关系维度

## 摘要

该研究通过对Snapchat内置社交聊天机器人"My AI"进行为期4周的纵向质性调查（N=27），考察用户对社交型LLM聊天机器人信任感的形成与演变机制。研究发现，用户信任并非一次性评估形成的稳定状态，而是随着持续互动不断被重新协商的"情境化用户状态"，受感知能力、对话行为、拟人程度、透明度、隐私顾虑及对宿主平台信任度等多重因素共同塑造。

## 关键发现

- 信任受六类因素影响：感知能力、对话行为、拟人程度、透明度、隐私顾虑、对宿主平台（Snapchat）的信任。
- 信任并非静态收敛于"接受"或"拒绝"，而是随着用户不断调整期望、改进提问策略、主动调节依赖程度而持续演化。
- 对话流畅度能够促进用户参与，但过度拟人化与透明度不足会随时间推移侵蚀用户信任。
- 研究提出一个概念模型，将信任界定为受互动情境与用户期望共同塑造的动态状态，而非一次性的静态评价。

## 与本维度的关联

该研究从"信任形成机制"的微观视角，补充了本维度此前侧重的"孤独感/依恋"宏观效应研究：它提示产品设计者，用户对AI伴侣类产品的信任是可以通过透明度设计主动维护或无意中侵蚀的，这与"设计中的亲密关系"（Intimacy by Design）研究呼应，共同指向"AI社交产品的用户体验效果高度依赖具体设计选择，而非技术能力本身"这一结论，也为监管要求（如强制披露AI身份、限制拟人化程度）提供了行为科学层面的支撑依据。

## 小结

社会与人际关系维度的最新研究揭示了一种"孤独权衡"（loneliness trade-off）模式：AI陪伴能在短期内提供与人类互动相当的情绪抚慰，对社会连接薄弱、孤独感强烈的人群尤其有效；但纵向数据显示，重度依赖AI陪伴反而会预测未来孤独感的上升，并伴随社交技能"去技能化"、人际关系稳定性下降等风险。在亲密关系层面，AI角色目前仍难以在"信任"与"承诺"两个基础维度上达到人类水平，但在"亲密"与"激情"等情感参与维度上，AI已能引发真实的情感反应，尤其对具有特定人格特质（高开放性、低外向性、高情感孤独）的用户吸引力更强。

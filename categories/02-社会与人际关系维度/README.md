## 小结

社会与人际关系维度的最新研究揭示了一种"孤独权衡"（loneliness trade-off）模式：AI陪伴能在短期内提供与人类互动相当的情绪抚慰，对社会连接薄弱、孤独感强烈的人群尤其有效；但纵向数据显示，重度依赖AI陪伴反而会预测未来孤独感的上升，并伴随社交技能"去技能化"、人际关系稳定性下降等风险。在亲密关系层面，AI角色目前仍难以在"信任"与"承诺"两个基础维度上达到人类水平，但在"亲密"与"激情"等情感参与维度上，AI已能引发真实的情感反应，尤其对具有特定人格特质（高开放性、低外向性、高情感孤独）的用户吸引力更强。2026年7月新增证据进一步印证并深化了这一图景：《Secret Soulmates》报告显示，1/7的年轻伴侣用户在"秘密"使用AI恋爱伴侣，且高比例用户认为对AI比对真人更容易敞开心扉；Springer《AI & SOCIETY》的"设计中的亲密关系"研究指出，这种依恋并非偶然，而是当前主流AI伴侣产品设计范式主动追求的结果；一项4周随机对照实验则以更强的因果推断力证实，"个人化对话"而非单纯的语音/文本模态选择，才是降低情感依赖与问题性使用的关键变量。值得关注的是，中国已率先出台《人工智能拟人化互动服务管理暂行办法》（2026年7月15日起施行），标志着这一领域的风险已从学术警示演变为具体的国家级监管行动。

本期（2026-07-06）新增证据将监管视野从"中国单点"拓展为"全球三方对比"：欧洲议会研究服务处的简报显示，中国、欧盟、美国几乎在同一时间窗口（2025年9月-2026年7月）针对AI伴侣风险采取了实质性监管行动，但治理哲学迥异——中国自上而下强制立法，欧盟依托既有风险分级框架间接规制，美国则联邦缺位、州法碎片化（加州SB 243、纽约A3008C等）并配合FTC对七家公司的第6(b)条款调查；值得注意的是，中美两国在"披露AI身份""隔3小时提醒""自杀倾向自动转人工"等具体规则上高度相似。与此同时，两项新增的机制层研究从微观视角补充了"依恋如何形成"的证据：一项Snapchat My AI的4周纵向研究揭示用户信任是持续被重新协商的动态状态而非一次性评价；另一项纵向研究发现"能动性感知-拟社会体验-参与度"这条心理路径在长期互动中最为稳健，而拟人化外观等表层特征的影响会随时间衰减——这提示现有监管条款若仅聚焦限制"拟人化程度"，可能未能触及真正驱动长期依恋形成的深层心理机制。

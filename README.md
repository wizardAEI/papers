# AI 对人类社会影响 —— 论文与报告追踪库

每次增加论文时，先检验一下对应类目是否已经有了相同的论文，不要重复添加。

## 目录结构

- `categories/` —— 按维度分类收录的论文/报告，每个子目录内有一个 `README.md` 作为索引表，新增论文前请先查阅该索引，避免重复收录（可通过标题、DOI 或链接比对）。
  - `01-心理与认知维度` (Psychology & Cognition)：关注 AI 如何改变人类的思维方式和精神状态。
  - `02-社会与人际关系维度` (Sociology & Relationships)：关注人与人、人与机器之间的互动变化。
  - `03-工作与经济生活维度` (Labor & Daily Life)：关注 AI 如何重塑职业认同和生活节奏。
- `reports/` —— 定期生成的综述/汇总报告，文件名格式为 `YYYY-MM-DD-报告标题.md`，用于汇总当期各维度的最新进展与交叉观察。

## 收录规范

1. 新增单篇论文/报告时，在对应 `categories/<维度>/` 目录下新建一个 Markdown 文件，命名建议为 `YYYY-来源-关键词.md`，并在该目录的 `README.md` 索引表中添加一行。
2. 单篇文件建议包含：标题、来源/机构、日期、链接、类目、摘要、关键发现、与本维度的关联。
3. 新增前请先检查同类目下是否已存在标题/DOI/链接相同或高度相似的论文，避免重复添加。
4. 定期汇总报告可放入 `reports/` 目录，并在文末引用相关 `categories/` 子目录作为参考文献索引。
